# biosense-hub — Project Plan / Goals

> ⚠️ Living document — rough plan, subject to change at any time.

## Goal
Personal health & environment dashboard — correlating room climate (ESP32-S3 + DHT22) with biometric data (Fitbit Inspire 3) via ASP.NET Core, SQLite and Blazor.

## Planned Features
- Live temperature & humidity readings from the ESP32
- Historical climate data over time (charts)
- Fitbit metrics (e.g. steps, heart rate, sleep) — to be defined
- Correlation view: climate vs. biometrics
